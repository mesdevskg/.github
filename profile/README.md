## 🏛️ Об организации
**«Окуу Китеби»** — государственная организация при Министерстве образования и науки Кыргызской Республики. Мы являемся ключевым звеном в создании, экспертизе и цифровизации учебных материалов нового поколения.

## 🎯 Основные направления
*   **Цифровая трансформация:** Разработка платформ для доступа к электронным учебникам (online/offline).
*   **Экспертиза:** Глубокая оценка качества и содержания образовательного контента.
*   **Издательство:** Координация создания современных учебников и методических пособий.
*   **Развитие системы:** Внедрение ИТ-инструментов для оценки знаний и поддержки школ КР.
