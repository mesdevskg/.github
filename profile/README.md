# 📚 Окуу Китеби

Платформа для экспертизы, анализа и улучшения учебных материалов.

---

## 🚀 О проекте

**Окуу Китеби** — это цифровая система (АИС Эксперт), которая помогает:
- проводить экспертизу учебников
- улучшать качество образовательных материалов
- обеспечивать прозрачность проверки

---

## 🎯 Наша цель

Повысить качество образования через:
- научную экспертизу
- педагогическую оценку
- современные цифровые инструменты

---

## 🧠 Виды экспертизы

- 🟢 Научная
- 🔵 Научно-педагогическая
- 🟣 Гендерная и антидискриминационная
- 🟠 Практико-педагогическая
- 🟤 Адаптированная (для зарубежных материалов)

---

## 🛠 Технологии

- Frontend: Next.js / React
- Backend: NestJS
- Database: PostgreSQL
- ORM: Prisma

---

## ⚙️ Возможности платформы

- 📂 Управление файлами и папками
- 📝 Комментарии и замечания к документам
- 📊 Отслеживание прогресса экспертизы
- 🔔 Уведомления в реальном времени
- 👥 Роли пользователей (эксперты, админы)

---

## 🌍 Миссия

Создать единую цифровую среду для экспертов образования в Кыргызстане и за его пределами.
